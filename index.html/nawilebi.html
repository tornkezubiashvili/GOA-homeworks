motherboard

The motherboard is the main part of the system unit that defines
the architecture and performance of the computer. It is also called the
system board, main or main board. It connects various devices to the processor
and regulates the pulses coming from these devices.

RAM

RAM is a temporary storage of information, in which data and programs to be processed by
the processor are written. The higher the amount of RAM and the clock frequency,
the higher the performance of the computer.

video card

Video card - a component of a computer whose purpose is to process graphic information and display
it on a monitor. Information processing is carried out by a graphics processing unit (GPU).
The number of cores of a graphics processor and the amount of video memory
have a great impact on the performance of this device.
